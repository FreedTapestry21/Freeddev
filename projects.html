Foxite

the only half-working web server you'll ever need.

Foxite is an open-source, kinda working web server made with Python.
Basically what Foxite does is it processes HTTP requests and responds with a webpage.
Foxite is made from the ground up, only using Python's socket, json, os and sys libraries.
